Material type

The `Material` type is used to control shaders in rendering. For a list of shader parameters, see https://developer.valvesoftware.com/wiki/Category:List_of_Shader_Parameters For a list of $flags and $flags2, see https://developer.valvesoftware.com/wiki/Material_Flags

Material:setTextureURL( key, url, cb, done )

Loads an online image or base64 data to the specified texture key

Parameters

  • key ― The key name to set. $basetexture is the key name for most purposes.
  • url ― The url or base64 data
  • cb ― An optional callback called when image is loaded. Passes nil if it fails or Passes the material, url, width, height, and layout function which can be called with x, y, w, h to reposition the image in the texture
  • done ― An optional callback called when the image is done loading. Passes the material, url
