Bass type

For playing music there is `Bass` type. You can pause and set current playback time in it. If you're looking to apply DSP effects on present game sounds, use `Sound` instead.

Bass:getFFT( n )

Perform fast Fourier transform algorithm to compute the DFT of the sound channel.

Parameters

  • n ― Number of consecutive audio samples, between 0 and 7. Depending on this parameter you will get 256*2^n samples.

Returns

Table containing DFT magnitudes, each between 0 and 1.

Bass:setFade( min, max )

Sets the fade distance of the sound in 3D space. Must have `3d` flag to get this work on.

Parameters

  • min ― The channel's volume is at maximum when the listener is within this distance
  • max ― The channel's volume stops decreasing when the listener is beyond this distance.
