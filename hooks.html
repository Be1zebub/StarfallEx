  • when ― The timer.realtime() at which this event occurred.
xinputPressed( id, button, when )

Called when a controller button has been pressed. Client must have XInput Lua binary installed.

Parameters

  • id ― Controller number. Starts at 0
  • button ― The button that was pushed. See https://github.com/mitterdoo/garrysmod-xinput#xinput_gamepad_
  • when ― The timer.realtime() at which this event occurred.
xinputReleased( id, button, when )

Called when a controller button has been released. Client must have XInput Lua binary installed.

Parameters

  • id ― Controller number. Starts at 0
  • button ― The button that was released. See https://github.com/mitterdoo/garrysmod-xinput#xinput_gamepad_
  • when ― The timer.realtime() at which this event occurred.
xinputStick( id, stick, x, y, when )

Called when a stick on the controller has moved. Client must have XInput Lua binary installed.

Parameters

  • id ― Controller number. Starts at 0
  • stick ― The stick that was moved. 0 is left
  • x ― The X coordinate of the trigger. -32768 - 32767 inclusive
  • y ― The Y coordinate of the trigger. -32768 - 32767 inclusive
  • when ― The timer.realtime() at which this event occurred.
xinputTrigger( id, trigger, value, when )

Called when a trigger on the controller has moved. Client must have XInput Lua binary installed.

Parameters

  • id ― Controller number. Starts at 0
  • trigger ― The trigger that was moved. 0 is left
  • value ― The position of the trigger. 0-255 inclusive
  • when ― The timer.realtime() at which this event occurred.
