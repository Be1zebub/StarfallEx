Functions definition

wire.adjustInputs( names, types )

Creates/Modifies wire inputs. All wire ports must begin with an uppercase letter and contain only alphabetical characters.

Parameters

  • names ― An array of input names. May be modified by the function.
  • types ― An array of input types. Can be shortcuts. May be modified by the function.
wire.adjustOutputs( names, types )

Creates/Modifies wire outputs. All wire ports must begin with an uppercase letter and contain only alphabetical characters.

Parameters

  • names ― An array of output names. May be modified by the function.
  • types ― An array of output types. Can be shortcuts. May be modified by the function.
wire.create( entI, entO, inputname, outputname, width, color, material )

Wires two entities together

Parameters

  • entI ― Entity with input
  • entO ― Entity with output
  • inputname ― Input to be wired
  • outputname ― Output to be wired
  • width ― Width of the wire(optional)
  • color ― Color of the wire(optional)
  • material ― Material of the wire(optional), Valid materials are cable/rope, cable/cable2, cable/xbeam, cable/redlaser, cable/blue_elec, cable/physbeam, cable/hydra, arrowire/arrowire, arrowire/arrowire2
