net.receive( name, func )

Like glua net.Receive, adds a callback that is called when a net message with the matching name is received. If this happens, the net hook won't be called.

Parameters

  • name ― The name of the net message
  • func ― The callback or nil to remove callback. (len - length of the net message, ply - player that sent it or nil if clientside)
net.send( target, unreliable )

Send a net message from client->server, or server->client.

Parameters

  • target ― Optional target location to send the net message.
  • unreliable ― Optional choose whether it's more important for the message to actually reach its destination (false) or reach it as fast as possible (true).
